# Customer Shopping Behavior Analysis

A concise analysis of customer shopping patterns built from transaction data — includes EDA, SQL queries, and a dashboard snapshot.

- **Dataset:** `customer_shopping_behavior.csv`
- **Deliverables:** `Customer_behavior_analysis.ipynb` (analysis notebook), `Customer_behavior_analysis.sql` (queries), dashboard snapshot below
- **Tech:** Python (pandas, plotly/matplotlib), Jupyter, SQL
- **How to view:** open the notebook or view the dashboard image in this repo

![Dashboard](./Screenshot%202026-03-04%20101207.png)

For details and reproducibility, open `Customer_behavior_analysis.ipynb`.
